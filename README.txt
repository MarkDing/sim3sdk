* SiM3 SDK 
* USB stack base on LUFA
* FAT 12/16/32 support base on Fatfs
* SD card driver support.
* MCU platform is SiM3U167 Arm Cortex M3 core.

Status
----------------------------------
* USB CDC class has been tested and works well.
* Other classes need to be tested in future.

* SD card SPI driver was done, not implement DMA support yet.
* FATFS read/write file and directory list work OK.
